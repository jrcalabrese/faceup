# faceup

> Load, organize, and wrangle OpenFace output.

## Install package

``` r
devtools::install_github("jrcalabrese/faceup")
library(faceup)
```

## Introduction

The **faceup** package is designed to make using [OpenFace](https://github.com/TadasBaltrusaitis/OpenFace) in research easier.
